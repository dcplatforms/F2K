# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-07-17

### Added
- Initial release of F2K (Farm-to-Table Kiosk)
- Core ecommerce functionality with React + TypeScript
- Viar Farms product catalog integration
- Shopping cart with quantity management
- Admin dashboard for inventory and order management
- Recipe hub with Gemini AI integration for personalized recommendations
- Responsive UI with Tailwind CSS
- GitHub Pages deployment configuration
- Complete documentation (CONTRIBUTING.md, CODE_OF_CONDUCT.md, DEVELOPMENT.md)
- CI/CD pipeline with GitHub Actions
- Branch protection rules on main branch

### Security
- Environment variable configuration for sensitive API keys
- Code owner review requirements for branch protection
- Stale review dismissal enabled for main branch
